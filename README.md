# BooneINC-Beta2
Project 2 for our team in ENGR1340

Proposal:
Boone Inc. is a website where users can use an interactive Calendar and To do list to plan out their weeks. This website’s target audience is students in college needing to allot their time to complete assignments and prepare for exams. The site aims to be easy to use, taking similar formats to applications like Apple notes’s task feature or Google Calendar. We also plan to feature an interactive mascot character named Boone which users can hover their mouse cursor over to see different speech bubbles appear with lines of dialogue. 

App Name:
Boone Inc.
App Type:
Calendar/Planner/To do list
Problem It Solves:
This app addresses the challenges for students to stay organized in school by combining the features we found the students we interviewed found most useful. The target audience for this project is college students that need to organize their schedules and plan out their time for assignments.


Page 1 — Name & Description:
Landing Page - Introduction screen featuring links to the calendar screen and the to do list screen
Page 2 — Name & Description:
Calendar page - Interactive and responsive calendar similar to Google Calendar allowing users to add and allot times for different tasks. It displays one week of plans at a time. 
Page 3 — Name & Description:
To-Do List Page/ Weekly Schedule - Modifiable to-do list that allows a user to organize tasks by importance
Key Interactive Feature 1:
Interactive mascot characters that users can hover mouse over or tab to and a speech bubble with new dialogue will appear.
Key Interactive Feature 2:
In the interactive responsive calendar, users can add multiple events and assign them a  time and name.
Key Interactive Feature 3:
The landing page features two interactive links to the other pages of the site.
Key Interactive Feature 4:
The Calendar and To Do list pages feature back buttons linking to the landing page.
